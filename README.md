# ToDoAppIos
I developed ToDo App with Swift and Sqlite for IOS. User can add a new task and delete or update the task. In addition, user can search. 

### Technologies
- [x] VIPER Pattern
- [x] SQLite
- [x] Tableview
- [x] Navigation
- [x] Auto Layout
- [x] Date picker
